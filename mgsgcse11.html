Systems Architecture

Component 01: Computer Systems

Overview

This section covers the fundamental components of a computer system and how they interact. Understanding systems architecture is crucial for comprehending how software instructions are executed by hardware.

Key Concepts

  • The purpose of the Central Processing Unit (CPU)
  • Von Neumann architecture: MAR, MDR, Program Counter, Accumulator
  • The function of the CPU components: ALU, CU, Cache
  • The Fetch-Decode-Execute cycle
  • Factors affecting CPU performance: clock speed, cores, cache size
  • Embedded systems: purpose and examples
