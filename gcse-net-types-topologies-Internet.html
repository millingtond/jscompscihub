The Internet, WWW, and How We Access It

The InternetA global network of interconnected networks connecting millions of computers, making it possible to exchange information. It is an example of a WAN. is a vast, global network of interconnected computer networks. It's the underlying infrastructure that allows computers all over the world to communicate with each other.

The World Wide Web (WWW)Consists of a collection of resources (like web pages, images, videos) accessible over the internet using protocols like HTTP., often just called "the web," is one of the many services that run on the Internet. It's a collection of websites, web pages, and online resources that are linked together by hyperlinks and accessed using URLsA location or address identifying where documents can be found on the Internet; a Web address. It includes the protocol and the domain name. (web addresses).

When you want to visit a website, your computer (the clientA device (e.g., computer, smartphone) or software (e.g., web browser) that requests services or resources from a server.) needs to find the web serverA computer that stores website files and serves them to clients (browsers) upon request. where that website's files are stored (hostedStoring website files on a web server, making them accessible via the Internet.). This process involves several key technologies, including the Domain Name System (DNS)A server that translates human-readable domain names (e.g., www.google.com) into numerical IP addresses that computers use to locate each other..

Real-World Internet Applications

  • E-commerce: Online shopping websites (Amazon, eBay) are hosted on web servers, use DNS for access, and often leverage cloud services for scalability and data storage.
  • Social Media: Platforms like Instagram and TikTok rely heavily on web servers to deliver content, DNS for user access, and massive cloud infrastructure to store user data, photos, and videos.
  • Online Banking: Secure web servers, DNS, and robust cloud or private server infrastructure are essential for providing safe online banking services.
  • Streaming Services (Netflix, Spotify): These use a global network of web servers (often Content Delivery Networks - CDNs, a form of distributed hosting) to stream media to users, all accessed via DNS and the internet.
  • Collaborative Tools (Google Docs, Microsoft 365): These are cloud-based applications, allowing multiple users to work on documents stored and processed on remote servers.

Internet & Cloud Myth Busters!

Myth: "The Internet and the World Wide Web are the same thing."

Reality: The InternetA global network of interconnected networks connecting millions of computers, making it possible to exchange information. It is an example of a WAN. is the global network infrastructure. The World Wide WebConsists of a collection of resources (like web pages, images, videos) accessible over the internet using protocols like HTTP. is a service that runs *on* the Internet, allowing access to web pages and other resources via URLs and protocols like HTTP.

Myth: "DNS servers always know every IP address for every website."

Reality: DNSA server that translates human-readable domain names (e.g., www.google.com) into numerical IP addresses that computers use to locate each other. is a hierarchical system. If your local DNS server doesn't know an IP address, it queries other DNS servers higher up in the hierarchy until the address is found or it's determined not to exist. They also cache results for a period to speed up future requests.

Myth: "Data stored in The CloudRemote servers and data storage accessed over the Internet, providing services like file storage, software applications, and processing power. is floating in the sky."

Reality: "The Cloud" refers to a network of powerful physical servers located in data centers around the world. When you store data "in the cloud," it's actually being saved on these remote hard drives, accessible via the internet.

Mark Scheme (Any two from):

  • Accessibility: Files can be accessed from any device with an internet connection. (1 mark)
  • Backup: Data is often automatically backed up by the provider, reducing risk of local data loss. (1 mark)
